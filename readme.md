1.俩类数据冒险：

sub x2, x1, x3

and x12, x2, x5

or  x13, x6, x2

-> sub & and == 1a

-> sub & or  == 2b 

### 机器码
`sub x2,x1,x3` 机器码：

0100000_00011_00001_000_00010_0110011

`and x12, x2, x5` 机器码：

0000000_00101_00010_111_01100_0110011

`or x13, x6, x2`

0000000_00010_00110_110_01101_0110011


### 首先赋初值
`addi x1, x0, 5`

`addi x3, x0, 3`

`addi x5, x0, 3`

`addi x6, x0, 5`

0000_0000_0101_00000_000_00001_0010011

0000_0000_0011_00000_000_00011_0010011

0000_0000_0011_00000_000_00101_0010011

0000_0000_0101_00000_000_00110_0010011

x2 = 2 -> 0010
x6 = 5 -> 0101

           0010	             0010
	&   0011         |   0101
	-----------      ----------
	    0010            0111


**验证检查：**

x13 = 7

整体汇编如下：

	addi x1, x0, 5
	addi x3, x0, 3
	addi x5, x0, 3
	addi x6, x0, 5
	sub x2, x1, x3
	and x12, x2, x5
	or  x13, x6, x2  


测试： PASS
